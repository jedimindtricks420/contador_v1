# Дизайн-система Contador v2.0

**Contador** — это ориентированная на продуктивность среда. В основе дизайна лежит принцип **отсутствия лишнего**, что позволяет пользователю сфокусироваться на числах и финансовых операциях.

## 1. Визуальный язык
- **Стиль:** Монохромный («Notion-style»). 
- **Палитра:**
  - `bg-white` (Основной фон)
  - `text-gray-900 / bg-black` (Основные акценты)
  - `border-gray-200 / bg-gray-50` (Разделение блоков)
- **Типографика:** `font-sans` (системные шрифты) с выраженным использованием `uppercase tracking-widest` для заголовков, чтобы подчеркнуть статусность Enterprise-решения.

## 2. Архитектура Адаптивного Смещения (Dynamic Offset)
При открытии ИИ-помощника срабатывает уникальная система перераспределения пространства:
1. **Состояние:** При `isChatOpen === true` макету `main` добавляется `padding-right: 400px`.
2. **Результат:** Вместо того чтобы перекрыть контент чатом, макет мягко сужается.
3. **Symmetry:** Использование `max-w-7xl mx-auto` внутри смещенного макета автоматически выравнивает контент по центру **нового** свободного пространства.

## 3. Компоненты UI
- **Sidebar:** Фиксированная панель слева (256px), содержит навигацию и Organization Switcher.
- **Journal Table:** Таблица с `font-mono` для сумм, обеспечивающая читаемость чисел в колонках.
- **Organization Switcher:** Компактный выпадающий список в нижней части сайдбара для быстрой смены контекста организации.

## 4. Контроль Фокуса
- Активные элементы ввода в Журнале подсвечиваются строгой черной рамкой (`focus:border-black`).
- Все действия (сохранение, удаление) защищены модальными подтверждениями, чтобы минимизировать ошибки.
