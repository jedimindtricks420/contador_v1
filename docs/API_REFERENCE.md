# Справочник API (v2.0)

Все запросы к API Contador защищены и требуют активной сессии пользователя.

## 1. Организации (Organizations)
- **`GET /api/organizations`**
  - Описание: Возвращает список всех компаний текущего пользователя.
- **`POST /api/organizations`**
  - Описание: Создание новой организации. Требует `name` и `inn`.
- **`DELETE /api/organizations/[id]`**
  - Описание: Полное удаление организации и всех связанных данных.

## 2. Счета (Accounts)
- **`GET /api/accounts`**
  - Описание: Список активных счетов текущей огранизации.
- **`POST /api/accounts`**
  - Описание: Активация счета из мастер-плана для текущей организации.
- **`PATCH /api/accounts/[id]/toggle`**
  - Описание: Включение/выключение счета. Включает валидацию на наличие операций.

## 3. Транзакции (Transactions)
- **`GET /api/transactions`**
  - Описание: Получение журнала проводок текущей организации.
- **`POST /api/transactions`**
  - Описание: Создание новой двойной проводки (Дебет/Кредит).
- **`DELETE /api/transactions/[id]`**
  - Описание: Мягкое удаление операции (`is_deleted: true`).

## 4. Настройки и Модули
- **`POST /api/settings`**
  - Описание: Обновление учетной политики (дата закрытия периода, дата открытия).
- **`GET /api/reports/opening-balance`**
  - Описание: Получение текущего статуса баланса по счету 0000.
