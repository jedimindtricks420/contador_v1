# GORGEOUS PARTNERS Accounting System (MVP)

Это веб-приложение для управленческого и бухгалтерского учета IT-сервисной компании. Система заменяет Excel, обеспечивая целостность данных через принцип двойной записи.

## Быстрый старт (Docker)

Приложение полностью контейнеризировано.

1. **Запуск:**
   ```bash
   docker-compose up -d
   ```
2. **Адрес:** `http://localhost:3030`
3. **Авторизация (Basic Auth):**
   - **Логин:** `admin`
   - **Пароль:** из файла `.env` (по умолчанию `securepassword`)

## Основные возможности

- **Дашборд:** KPI, графики выручки и расходов.
- **Журнал:** Ввод операций, мягкое удаление, контроль закрытого периода.
- **Остатки (1C-style):** Ввод начальных остатков через технический счет 0000 с датой открытия и фиксацией.
- **Отчеты:** ОСВ (с S1, оборотами и S2), Баланс (Форма 1), ОФР (Форма 2).
- **Контроль:** Блокировка редактирования закрытых периодов и остатков.
- **Контрагенты:** Единый справочник партнеров.
