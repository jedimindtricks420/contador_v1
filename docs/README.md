# CONTADOR v2.0 — Enterprise Accounting System

**Contador** — это мощная SaaS-система для бухгалтерского и управленческого учета, построенная на принципах НСБУ №21 и адаптированная для работы с несколькими организациями (Multi-tenancy). 

Система заменяет традиционные Excel-таблицы, обеспечивая строгую целостность данных через принцип двойной записи и изолированные пространства для каждого юридического лица.

## Ключевые возможности v2.0

- **Multi-tenancy & Auth:** Полноценная регистрация пользователей и управление неограниченным количеством организаций в одном профиле.
- **Enterprise Onboarding:** Умный 3-х шаговый процесс создания компании с выбором из 22 отраслевых шаблонов (IT, Торговля, Производство и др.).
- **План счетов НСБУ:** Предустановленные ~800 мастер-счетов с возможностью гибкой активации и настройки под специфику бизнеса.
- **AI-Помощник:** Глобальный ассистент для анализа проводок и помощи в ведении учета.
- **Финансовая отчетность:** Автоматическая генерация ОСВ, Баланса (Форма 1) и ОФР (Форма 2) в реальном времени.
- **Контроль периодов:** Блокировка редактирования операций в закрытых периодах и фиксация начальных остатков (счет 0000).

## Быстрый старт (Docker)

Приложение полностью контейнеризировано и готово к работе:

1. **Запуск:**
   ```bash
   docker-compose up -d --build
   ```
2. **Адрес:** `http://localhost:3030`
3. **Первый вход:** 
   - Зарегистрируйтесь на странице `/register`.
   - Система автоматически направит вас в Онбординг для создания первой организации.
